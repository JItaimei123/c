# 函数递归

## 什么是递归？

递归是一种编程技巧，函数调用自身。

## 举例1：阶乘

```C
int Fact(int n) {
    if (n <= 1) return 1;
    return n * Fact(n - 1);
}
```

## 举例2：顺序打印一个整数的每一位

```C
void print(int n) {
    if (n > 9) {
        print(n / 10);
        printf("%d ", n % 10);
    } else {
        printf("%d ", n);
    }
}
```

思路：`1234/10 == 123`，`1234%10 == 4`

## 递归与迭代

递归层次太深会浪费栈帧空间，可能引起**栈溢出（stack overflow）**。

此时可以用**迭代**（循环）代替。

### 阶乘（迭代版）

```C
int Fact(int n) {
    int ret = 1;
    for (int i = 1; i <= n; i++) {
        ret *= i;
    }
    return ret;
}
```

### 斐波那契数列

递归版会有大量重复计算，效率极低。

```C
int Fib(int n) {
    if (n <= 2) return 1;
    return Fib(n - 1) + Fib(n - 2);
}
```

迭代版：

```C
int Fib(int n) {
    int a = 1, b = 1, c = 1;
    while (n >= 3) {
        c = a + b;
        a = b;
        b = c;
        n--;
    }
    return c;
}
```
