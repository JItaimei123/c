# 数据类型与变量

## 布尔类型

```C
#include<stdbool.h>
int main() {
    bool islike = true;
    if (islike) printf("i like c\n");
    else printf("i don't like\n");
    return 0;
}
```

## sizeof操作符

```C
printf("%zd\n", sizeof(int));      // 4
printf("%zd\n", sizeof(char));     // 1
printf("%zd\n", sizeof(short));    // 2
printf("%zd\n", sizeof(long));     // 4
printf("%zd\n", sizeof(float));    // 4
printf("%zd\n", sizeof(double));   // 8
printf("%zd\n", sizeof(long long)); // 8
printf("%zd\n", sizeof(bool));     // 1
```

- sizeof 结果用 `%zd` 打印
- 表达式中不参与实际运算，只根据类型得大小
- 低精度向高精度转换

## signed与unsigned

- `signed`：有符号，可正可负
- `unsigned`：无符号，非负整数，**同样内存能表示的最大值翻倍**

## 变量

### 变量的分类

- **全局变量**：大括号外部定义，整个工程可用
- **局部变量**：大括号内定义，只在局部范围可用
- 内存分区：**栈区**（局部变量/函数参数）、**堆区**（动态内存）、**静态区**（全局/静态变量）

### 重名规则
局部内优先打印局部变量，否则打印全局变量。

## 算术操作符

- `+ - *`：双目操作符
- `/`：整数除法得整数，浮点数除法需至少一个操作数为浮点数
- `%`：求余，只能用于整数，结果正负由第一个操作数决定

## 赋值操作符

### 连续赋值
```C
c = b = a + 3; // 从右向左
```

### 复合赋值符
`+=` `-=` `*=` `/=` `%=` `>>=` `<<=` `&=` `|=` `^=`

## 单目操作符

### 前置++
```C
int b = ++a; // a=a+1, b=a
```

### 后置++
```C
int b = a++; // b=a, a=a+1
```

## 强制类型转换

```C
int a = (int)3.14;
```

## scanf与printf

### printf
- `%d` 整数, `%f` float, `%lf` double, `%c` 字符, `%s` 字符串, `%p` 地址, `%zd` size_t
- 可限定宽度：`%5d`, `%.2f`, `%6.2f`, `%-5d`（左对齐）
- 可显示正负号：`%+d`

### scanf
- 变量前必须加 `&`（数组名本身是地址除外）
- 返回值：成功读取的变量个数
- `%c` 不忽略空白字符，其他占位符自动忽略
- `%s` 从第一个非空白字符读到空白字符为止
- `%[m]s` 指定最大读取长度
- 赋值忽略符 `*`：`scanf("%d%*c%d%*c%d", &y, &m, &d)`
