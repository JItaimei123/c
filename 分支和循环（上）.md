# 分支和循环（上）

## 1. if语句

### 1.1 if
在C语言中，0为假，非0表示真。

```C
#include<stdio.h>
int main() {
    int num = 0;
    scanf_s("%d", &num);
    if (num % 2 == 1)
        printf("奇数\n");
    return 0;
}
```

### 1.2 else语句

```C
#include<stdio.h>
int main() {
    int num = 0;
    scanf_s("%d", &num);
    if (num % 2 == 1) printf("奇数\n");
    else printf("偶数\n");
    return 0;
}
```

### 1.3 分支中包含多条语句
默认if和else只控制一条语句，需要用 `{}` 括起来控制多条。

### 1.4 嵌套语句

```C
if (num == 0) printf("0\n");
else if (num > 0) printf("正数\n");
else printf("负数\n");
```

### 1.5 悬空else问题
else 总是跟最接近的 if 匹配。使用 `{}` 避免歧义。

## 2. 关系操作符

`>` `<` `>=` `<=` `==` `!=`

注意：`==` 与 `=` 不要混淆。运算符不宜连用如 `i < j < k`，应写成 `i < j && j < k`。

## 3. 条件操作符

`exp1 ? exp2 : exp3`

```C
int m = a > b ? a : b;
```

## 4. 逻辑运算符

- `!`：逻辑取反
- `&&`：逻辑与（并且）
- `||`：逻辑或（或者）

### 短路
- `&&`：左边为0时，右边不再执行
- `||`：左边非0时，右边不再执行

### 闰年判断

```C
if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0))
    printf("闰年\n");
```

## 5. switch语句

```C
switch (expression) {
    case value1: statement; break;
    case value2: statement; break;
    default: statement; break;
}
```

- expression 必须是整型表达式
- case 后必须是整型常量表达式
- break 跳出 switch，否则会继续执行下一个 case

## 6. while循环

```C
while (表达式) { 语句; }
```

### 逆序打印整数每一位

```C
while(n) { printf("%d ", n%10); n /= 10; }
```

## 7. for循环

```C
for(表达式1; 表达式2; 表达式3) { 语句; }
```

for的三个部分（初始化、判断、调整）集中在一起，便于维护。

### 计算1~100之间3的倍数的数字之和

```C
int sum = 0;
for(int i = 3; i <= 100; i += 3) sum += i;
```

## 8. do-while循环

```C
do { 语句; } while(表达式);
```

至少执行一次。

### 计算整数位数

```C
int cnt = 0;
do { cnt++; n /= 10; } while(n);
```

## 9. break和continue

- **break**：永久终止循环
- **continue**：跳过本次循环中 continue 后的代码

### while中的continue
跳过 `i = i+1` 可能导致死循环。

### for中的continue
跳到调整部分 `i++`，不会死循环。

## 10. 循环嵌套

### 找出100~200之间的素数

```C
for (num = 100; num <= 200; num++) {
    isPrime = 1;
    for (i = 2; i <= num / 2; i++) {
        if (num % i == 0) { isPrime = 0; break; }
    }
    if (isPrime) printf("%d ", num);
}
```

## 11. goto语句

在同一函数内跳转到标号处。在多层循环中快速跳出时很方便，但应谨慎使用。
