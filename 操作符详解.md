# 操作符详解

## 操作符分类

算术：`+ - * / %`
移位：`<< >>`
位：`& | ^ ~`
赋值：`= += -= *= /= %= <<= >>= &= |= ^=`
单目：`! ++ -- & * + - ~ sizeof (类型)`
关系：`> >= < <= == !=`
逻辑：`&& ||`
条件：`? :`
逗号：`,`
下标：`[]`
函数调用：`()`
结构成员：`. ->`

## 二进制和进制转换

### 2进制转10进制
每一位的权重是 2^n，从右向左。

### 10进制转2进制
除2取余法。

### 2进制转8进制
每3位二进制换算为1位八进制（2^3=8）。

### 2进制转16进制
每4位二进制换算为1位十六进制（2^4=16）。十六进制前加 `0x`。

## 原码、反码、补码

- **正数**：原码 = 反码 = 补码
- **负数**：原码 → 取反（符号位不变）→ 反码 → +1 → 补码
- 数据存放和运算用**补码**，打印用**原码**

```C
char a = 128;
printf("%d", a); // -128（符号扩展）
printf("%u", a); // 4294967168（高位补1作为数值）
```

## 移位操作符

|操作|规则|效果（正数）|
|-|-|-|
|`<<`|左边抛弃，右边补0|乘2|
|`>>`|右边抛弃，左边补符号位|除以2|

不要移动负数位。

## 位操作符

- `&`：按位与，有0则0
- `|`：按位或，有1则1
- `^`：按位异或，相同为0，相异为1（`a^a=0, 0^a=a`）
- `~`：按位取反

### 不创建临时变量交换两数

```C
a = a ^ b;
b = a ^ b;
a = a ^ b;
```

### 统计二进制中1的个数

```C
while (n) { n = n & (n - 1); count++; }
```

### 判断2的次方数

```C
n > 0 && (n & (n - 1)) == 0
```

## 逗号表达式

从左向右依次执行，整个表达式的结果是最后一个表达式的结果。

```C
int c = (a > b, a = b + 10, a, b = a + 1);
```

## 下标访问和函数调用

- `[]`：操作数是数组名和下标
- `()`：操作数是函数名和参数
