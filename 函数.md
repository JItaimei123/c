# 函数

## 函数的概念

C语言中的函数是一段完成某项特定任务的代码。C程序由无数个小的函数组合而成。

## 库函数

C标准规定了库函数的标准，不同编译器厂商实现。如 `printf`、`scanf`、`sqrt` 等。

库函数的使用需要包含对应的头文件。

## 自定义函数

```C
ret_type fun_name(形式参数) {
    // 函数体
}
```

### 加法函数示例

```C
int Add(int x, int y) {
    int z = 0;
    z = x + y;
    return z;
}
int main() {
    int a = 0, b = 0;
    scanf("%d %d", &a, &b);
    int r = Add(a, b);
    printf("%d\n", r);
    return 0;
}
```

## 形参和实参

- **实参**：调用函数时传递给函数的参数
- **形参**：定义函数时括号中的参数，只有被调用时才向内存申请空间

形参是实参的一份临时拷贝，各自独立的内存空间。

## return 语句

- 可以返回数值或表达式
- 返回类型为 void 时可写 `return;` 提前结束
- 返回的值和函数返回类型不一致时，系统会隐式转换
- return 执行后函数彻底返回

## 数组做函数参数

数组传参时退化为指针，需额外传递长度。

```C
void set_arr(int arr[], int sz) {
    for(int i = 0; i < sz; i++) arr[i] = -1;
}
```

## 嵌套调用和链式访问

### 嵌套调用
函数之间互相调用。

### 链式访问
将一个函数的返回值作为另一个函数的参数。

```C
printf("%d", printf("%d", printf("%d", 43)));
// 输出 4321，printf 返回打印字符的个数
```

## 函数的声明和定义

### 单个文件
在调用前声明函数：`int is_leap_year(int y);`

### 多个文件
- `.h` 头文件：存放函数声明、类型声明
- `.c` 源文件：存放函数实现

## static 和 extern

### static 修饰局部变量
改变生命周期：从栈区变为静态区，出函数不销毁。

### static 修饰全局变量
外部链接属性变为内部链接属性，只能在本源文件内使用。

### extern
声明外部符号，用于跨文件使用全局变量或函数。
